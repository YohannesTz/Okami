# Okami
An easy app for anime and manga exploration.

icons from [Icons8](https://icons8.com/)

# Features
- Display anime 
- Display manga 
- Share Manga/Anime

# Used technologies
- Dependency injection
- View Model
- Kotlin coroutines
- Flow
- Coil
- Retrofit
